&TEA x COFE x李君慈攜手黑金找茶 打造全台首款紅玉奶茶巧克力！

ICA金獎茶巧克力幻化新穎手搖 探索茶與巧克力無窮可能！

新銳設計鬼才操刀主視覺翻玩幾何圖樣12/15冬令限定派對開跑！

時序邁入聖誕時節，極簡手搖品牌&TEA擅長翻玩台灣茶茗與原物料開創與眾不同的新穎飲品風味，此次攜手ICA金獎巧克力職人COFE歷經一季反覆嘗試打造出兩款全台獨家的「紅玉奶茶巧克力」及「桂花鐵觀音茶巧克力」，&TEA研發主理人運用茶巧克力設計兩款冬令限定飲品，「瑞雪紅玉那堤」以經典紅茶拿鐵為概念混搭黑糖肉桂淋醬與COFE紅玉奶茶巧克力，「巧霜冬之花」則透過結合桂花鐵觀音茶底並鋪上巧克力奶霜及運用COFE桂花鐵觀音茶巧克力甘納許綴飾杯頂，為賦予本次合作展現更具生命力的作品，力邀新銳設計師李君慈擔綱本次合作主視覺及限定款杯套設計，以具象化幾何圖樣為概念，襯托出茶葉與可可共同演繹的冬日派對景象，三方合作大膽地突破消費者既定印象，聯名限定飲品將於12/15於&TEA敦北店限定發售，力邀消費者一同踏入聯名共舞的黑金派對！

ICA金獎茶巧克力結合法式輕奶油工藝，優雅轉身化作新穎肉桂捲風味手搖！

巧克力職人COFE以&TEA「瑞雪伯爵紅烤奶」及「巧霜鐵觀音」兩款經典飲品為概念發想，以可可脂低溫慢萃工法，將茶葉原始風味緩緩帶出並使用少量冰糖進行風味調和，獨賣款「紅玉奶茶巧克力」特別使用奶粉替換黃豆粉，逐步調整製程與添入時間，收斂奶粉原有苦味，賦予紅玉紅茶巧克力尾韻香醇甜郁的迷人奶香，品嚐時感受頂級魚池紅玉獨有的薄荷香氣在舌尖瀰漫，爾後奶香接踵而至共譜雙重協奏曲。「桂花鐵觀音茶巧克力」比照COTE茶巧克力的製成規格，以原生金珠黃豆粉穩定並柔化鐵觀音本身茶濃連帶澀感強的特點，精準調整入花時刻避免花香溢散，確保花香完整萃取於可可脂中並能與茶香相輔相成。

茶葉與可可幻化幾何物件，設計女力李君慈催生趣味橫生冬日找茶派對！

&TEA與COFE攜手聯名，以Cacao ParTEA黑金找茶作為核心理念，期望賦予雙品牌活潑與質感兼具的品牌意象，力邀新銳設計師李君慈擔綱聯名主視覺及限定造型杯套設計，她分享道，「接收到訊息後，就以主題parTEA 派對感、個性化具質感的方向著手，加上品質與品味這些都是從一開始接觸後理解的兩品牌合作想傳遞給消費者的氛圍。對於品牌而言每株茶葉、每分可可脂甚至是每顆糖粒都是匯聚成一杯飲品或是 一片巧克力不可或缺的核心要素，於是我將所有物件幾何化，將其派對熱鬧的氛圍用顏色去安排，讓每個元素都仿若是參加派對的來賓，也是派對上讓你的裝飾，讓看見圖像的人像進入一個充滿裝飾節慶氣氛的佈置。」

以半弧形方式呈現而標準字有秩序的錯落其中，就像將空間整個濃縮在杯身會場中，由世界各地匯聚而來的元素於此展開為期一季的在地體驗。即便在無法出國的時刻，也能透過品嚐聯名飲品、巧克力，感受仿若身處斯里蘭卡烏瓦茶園抑或是肯亞可可莊園意象。

「在杯套設計上將可可果實設定為螢光特別色，吸睛配色讓大家手拿時，便有匯聚路人目光 的特殊效果，如同你是派對上最閃亮的那顆星，而當三、四個杯套同時並排陳列時，能瞧見 派對的完整面貌，也是留給大家一同來找茶元素的小趣味！」

– &TEA x COFE x 李君慈聯名系列—

  • 瑞雪紅玉那堤 (M) NT$85 / (L) NT$90

基底選用CTC錫蘭紅茶搭配小農鮮奶製成醇厚紅茶拿鐵，鋪滿法式海鹽奶蓋後淋上秘製黑糖肉桂淋醬增添香氣，杯頂撒上COFE紅玉奶茶巧克力碎片。

  • 巧霜冬之花 (M) NT$85 / (L) NT$90

茶體選用散發淡淡焙火香氣鐵觀音，新穎巧克力奶霜輕鋪於杯頂，淋上COFE桂花鐵觀音茶巧克力甘納許與乾桂花增添香氣。
